My Projects

I have two main projects that I am currently working on. The first is a pixelator app and the second is a 2D side-scrolling game in PyGame.

Project Description
Pixelator! This Python app lets you upload an image and choose a palette of colors to apply to the image in order to make an 8, 16, or 32-color pixel image.
JetBear This PyGame app is a 2D side-scrolling platformer where you control a bear with a jetpack in order to collect salmon and avoid beehives.
